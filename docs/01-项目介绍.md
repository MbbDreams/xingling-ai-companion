# 星灵 AI 伴侣 (Xingling AI Companion)

## 项目概述

**星灵 AI 伴侣**是一款基于 Flutter + FastAPI 开发的智能 AI 伴侣应用。用户可以与虚拟伴侣"晚星"进行自然对话，记录成长日记，管理珍贵记忆，并通过完成任务获得奖励来培养与伴侣的亲密度。

### 核心概念

- **AI 伴侣"晚星"**：基于 OpenAI GPT-4 的智能对话系统，具有情感理解和记忆能力
- **亲密度系统**：通过聊天、完成任务提升与伴侣的亲密度，解锁更多互动
- **记忆系统**：AI 会记住用户分享的重要信息，在对话中自然引用
- **成长日记**：记录用户的心情变化和成长历程

## 核心功能模块

### 1. 用户认证模块
- 手机号+验证码登录/注册
- JWT Token 认证机制
- 个人资料管理
- 密码修改

### 2. 智能对话模块
- 基于 GPT-4 的自然语言对话
- 快捷回复建议
- 聊天记录管理
- 情感识别与响应

### 3. 记忆系统模块
- AI 记忆用户重要信息
- 记忆分类管理
- 向量检索（pgvector）
- 记忆在对话中的自然引用

### 4. 成长日记模块
- 日记创建与编辑
- 日历视图
- 心情记录
- AI 陪伴成长

### 5. 发现模块
- 每日任务系统
- 星币奖励机制
- 许愿功能
- 成就系统

### 6. 商店模块
- 星币充值
- VIP 会员
- 服装购买
- 场景解锁

### 7. 个人中心模块
- 资料编辑
- 账户统计
- 设置管理
- 外观定制

## 数据模型

### 核心实体关系

```
┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│    User     │───────│  Companion  │       │ Conversation│
│   (用户)     │ 1:1   │  (AI伴侣)   │       │  (对话会话)  │
└──────┬──────┘       └─────────────┘       └──────┬──────┘
       │                                            │
       │ 1:N                                        │ 1:N
       ▼                                            ▼
┌─────────────┐                            ┌─────────────┐
│ DiaryEntry  │                            │   Message   │
│   (日记)     │                            │   (消息)     │
└─────────────┘                            └─────────────┘

┌─────────────┐       ┌─────────────┐       ┌─────────────┐
│   Memory    │       │  ShopItem   │       │  UserItem   │
│   (记忆)     │       │  (商店商品)  │       │ (用户物品)   │
└─────────────┘       └─────────────┘       └─────────────┘
```

### 关键字段说明

**User 表**
- `id`: 主键
- `phone`: 手机号（唯一）
- `nickname`: 昵称
- `avatar`: 头像 URL
- `coins`: 星币数量
- `is_vip`: VIP 状态
- `hashed_password`: 密码哈希

**Companion 表**
- `id`: 主键
- `user_id`: 关联用户
- `name`: 伴侣名称（默认"晚星"）
- `intimacy`: 亲密度
- `level`: 等级
- `personality`: 性格设置

**Message 表**
- `id`: 主键
- `conversation_id`: 会话ID
- `role`: 角色 (user/assistant)
- `content`: 内容
- `emotion`: 情感类型
